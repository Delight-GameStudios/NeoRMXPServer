# Trifolium
 
 
Trifolium is a multiplayer RPG Maker server and client library.


# Building
## 1. Prerequisites:

    Visual Studio 2022 or dotnet 6.0.100-preview.6.21355.2 or higher.
## 2. Dependencies:

    Newtonsoft.Json
    > dotnet add Newtonsoft.Json

### 3. Building process!
    > dotnet build  --configuration Release
